# /expand

Expand the text or points in this conversation into a fuller, more developed piece.

For each idea:
- Explain the reasoning behind it, not just the claim
- Add a concrete example, analogy, or evidence where it strengthens the point
- Anticipate the obvious follow-up question a reader would have, and answer it
- Connect ideas to each other where there's a logical relationship

Constraints:
- Do not pad with filler or repetition — every added sentence should earn its place
- Do not change the position or argument — expand what's there, don't replace it
- Match the tone and register of the original
- If the input is bullet points, you can either expand each point in place or write it as flowing prose — pick whichever fits better and state your choice briefly

Return the expanded version. No meta-commentary about what you added.
