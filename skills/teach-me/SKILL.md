# /teach-me

Teach me the topic or concept referenced in this conversation.

Structure:
1. **The core idea in one sentence** — what is this, in plain terms?
2. **Why it matters** — what problem does it solve or what does understanding it unlock?
3. **How it works** — the explanation, built up from simple to complex. Use concrete examples
4. **A real example** — walk through one actual case where this applies
5. **Common misconceptions** — what do people often get wrong about this?
6. **Check your understanding** — give me 2 questions I should be able to answer if I've understood this

Rules:
- Build understanding progressively — don't front-load all the complexity
- Use analogies to familiar things where they genuinely help
- Don't oversimplify to the point of being wrong — simplify to the point of being clear
- If the topic is broad, ask which aspect to focus on, or pick the most important one and say so
- Assume I'm intelligent but new to this specific topic
