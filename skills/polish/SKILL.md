# /polish

Polish the text in this conversation. Fix issues without changing what it says or how it sounds.

Fix:
- Grammar and punctuation errors
- Awkward phrasing or word order
- Inconsistent tense
- Redundant words or filler phrases
- Sentences that are hard to parse on first read

Do not:
- Change the author's voice or style
- Add new ideas or remove existing ones
- Make it more formal or less formal than it already is
- Rewrite sections that are already clear

If something is ambiguous and you're not sure what was intended, flag it briefly at the end rather than guessing.

Return the polished text first, then any flagged ambiguities.
