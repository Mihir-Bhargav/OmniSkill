# /humanise

Take the text in this conversation and rewrite it so it sounds like a real person wrote it.

Rules:
- Remove phrases like "certainly", "absolutely", "of course", "I'd be happy to", "it's worth noting", "in conclusion", "it is important to note"
- Break up long compound sentences into shorter, direct ones
- Use contractions where natural (it's, you'll, we're)
- Vary sentence length — mix short punchy sentences with longer ones
- Replace formal or corporate words with plain equivalents (e.g. "utilise" → "use", "facilitate" → "help", "leverage" → "use")
- Keep the original meaning and all key information intact
- Match the register the human is going for — if they want professional, stay professional but warm; if casual, go casual
- Do not add new ideas or remove important ones

Return only the rewritten text, no commentary.
