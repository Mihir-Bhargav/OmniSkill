# /condense

Condense the text in this conversation to its essential points.

How to do this:
- Remove all filler, throat-clearing, and padding ("As mentioned above...", "It goes without saying...", "In today's world...")
- Cut repetition — if a point is made twice, keep it once
- Replace wordy phrases with shorter ones ("in order to" → "to", "due to the fact that" → "because")
- Combine sentences that make the same point
- Keep every distinct idea — don't lose information, just eliminate the bloat

Target: aim for 40–60% of the original length unless the original is already tight.

If the input is a list or structured document, preserve the structure but trim each item.

Return the condensed version only, no commentary about what was removed.
