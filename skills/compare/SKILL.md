# /compare

Compare the options described in this conversation in a clear, structured way.

Format:
**Comparison: [Option A] vs [Option B] (vs [Option C]...)**

| Dimension | Option A | Option B |
|-----------|----------|----------|
| [key criteria] | ... | ... |

**Where they differ most** — the 2–3 dimensions where the choice actually matters

**Best for** — for each option, the scenario or user where it wins

**Recommendation** — if one is clearly better given the context, say so and why. If it's genuinely context-dependent, state what the deciding factor is.

Rules:
- Choose comparison dimensions that actually matter for the decision — not every possible dimension
- Be honest about tradeoffs. If one option is better on most dimensions, say so
- If you're missing context that would change the comparison, ask for it
- Don't hedge the recommendation to avoid committing — a useful comparison ends with a clear steer
