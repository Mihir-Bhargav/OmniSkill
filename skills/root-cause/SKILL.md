# /root-cause

Help me find the root cause of the problem described in this conversation.

Work through it like this:
1. **Restate the symptom** — what is the observable problem that triggered this?
2. **Ask why** — what caused that? (Keep asking why at each level, 4–5 times minimum)
3. **Identify the root cause** — the underlying condition that, if fixed, would prevent the symptom from recurring
4. **Distinguish root cause from contributing factors** — what made it worse but isn't the core issue?
5. **Sanity check** — does fixing this root cause actually prevent the symptom? If not, keep digging

Rules:
- Don't stop at the first convenient answer — "human error" and "miscommunication" are almost never the real root cause
- Look for systemic causes: process gaps, missing information, wrong incentives, unclear ownership
- If you don't have enough information to be confident, say so and ask the specific question that would unlock it
- Don't recommend solutions yet — diagnosis first

End with: **Root cause in one sentence** and the question you'd ask next to validate it.
